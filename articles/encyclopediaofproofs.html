Encyclopedia of Proofs


By: Hayden Gemeinhardt | March 30, 2021

Courtesy to _Alicja_ @pixabay


So far at LSU, one of my favorite classes I have taken is Discrete Mathematics. There’s something about its complex simplicity, like a modern painting. And thus, I randomly decided to stay up to four A.M. creating an encyclopedia of its many proofs!

The fundamental basis lies in the “imply” principal (symbolized as ->). Simply put, P implies Q means Q is necessary for P (though P is not necessary for Q).
For example: If it is raining, then there are clouds in the sky.
Rain (P) is not necessary when there are clouds in the sky (Q). However, there must be clouds in the sky (Q) for there to be rain (P).
The implication is only false when it is raining (P) without clouds in the sky (/Q).

Types of Proofs:

Direct Proof- Assume P, explain how it concludes Q.


Theorem: For all n, if n is even, then n^2 is even.
Proof:
n=2k
n2 = (2k)2 = 4k2 = 2(2k2) = 2z where z = 2k2
Since z is divisible by 2, n2 must also be divisible by 2, and thus is even.


Trivial Proof: Show Q is always true regardless of P


Claim: If 0<n<1, then n2-2n+2 != 0
Proof:
(n-1)2 >=0 since every square is non negative.
(n-1)2 = n2-2n+1 >= 0
Adding 1 to both sides: n2-2n+2 >= 1 =! 0
In other words, no matter what n is chosen, the result can never be less than 1. Thus, the proposition 0


Vacuous Proof: Show P is always false regardless of Q


Remember, p->q is true when p is false. The “implies” statement is false only when p is true and q is false.
Claim: If n2+1<0, then n5>=4
Proof:
n2>=0 for all n (since every square is non negative),
then n2+1>=n^2>=0
Thus n2+1 cannot be less than 0. The proposition P is always false.


Proof By Contradiction


Assume /P, show /Q, then claim //P=P
Claim: Square root of 2 is irrational (cannot be written as a fraction)
Proof:
Before we continue, let’s clarify two things. For a/b to be irrational, a or b is odd, or both a and b is odd. And if n=2m, then n is even as it can be divided by 2 (n/2=m).
sqrt(2) = a/b we are assuming /P, or saying sqrt(2) can be written as a fraction
2 = (a/b)2 = a2/b2
2b2 = a2 Here, a2 must be even since it is divisible by 2, and thus b must be odd to satisfy irrationality (because a and b cannot both be even).
2b2 = (2c)2 Here, we can say a=2c since a is even, where c=a/2
2b2 = 4c2
b2 = 2c2 Here, b must be even because it is divisible by 2.
However, earlier we said b must be odd. Therefore, we have a contradiction.


Proof by Contrapositive: /Q->/P


Claim: if a+b is odd, then a is odd or b is odd.
Proof:
a is even and b is even (/Q).
a=2m and b=2n
a+b = 2m+2n = 2(m+n). Since it is divisible by 2, it must be even (/P)
Thus we showed /Q->/P


Proof by Exhaustion: Show all possibilities to be true


Claim: n+7>n2 for all integers 01+7=8>11=1
2+7=9>22=4
3+7=10>33=9


Proof by Case: (P1 or P2 or P3)->Q = (P1->Q and P2->Q and P3->Q)


Claim: n2>=n if n is an integer
Case 1: For n=0, 02>=0
Case 2: For n>=1, n2>=n then n*n>=n*1 then n>=1
Case 3: For n<0, all squares are positive, so n2>0>n


Constructive Proof by Existence: Show there exists


Claim: A positive integer can be written as the sum of cubes of a positive integer in two different ways
1729 = 103 + 93 = 123 + 13


Nonconstructive Proof by Existence: Prove there must exist


Claim: Show there exists an x where x3+x-5 = 0
Intermediate Value Theorem (IVT): If f(x) is continuous on the interval [a,b] and N is a value between f(a) and f(b), then there exists a number c such that f(c)=N
a=1, f(a) = -3
b=2, f(b) = 5
-3<0<5, thus by IVT, there is a number c such that f(c)=0


Uniqueness Proof: Element exists and is unique (2 parts)


Claim: There is a unique function f(x) such that f’(x)=2x and f(0)=4
Existence: x2+4
Uniqueness: Suppose f1(x) and f2(x) both satisfy these conditions. f1’(x) = f2’(x) = 2x, and thus must differ by some constant C (since the integral of 2x = x2+C) as there is nothing else for them to differ by without affecting the derivative. 3 = f1(0) = f2(0)+C = 3+C. C must equal 0 for 3=f(2)=3+C, which means f1(x)=f2(x), and therefore the function is unique because there is no C value for which to make the two equations different yet have the same derivative 2x AND have f(0)=3.


Proof by Induction: Prove true when n=1, assume it to be true for n=k (induction hypothesis), prove true for n=k+1


“I think some intuition leaks out in every step of an induction proof” Jim Propp
In computer science, we are met with this equation all the time: 1+2+…+n=n(n+1)/2
For n=1: 1=1*(1+1)/2=1 Good
Assume n=k: 1+2+…+k=k(k+1)/2 Induction Hypothesis
Show n=k+1 holds: 1+2+…+(k+1) = (k+1)*((k+1)+1)/2 = ((k+1)(k+2))/2
Starting with the induction hypothesis and trying to reach n=k+1, let’s add k+1 to both sides of the equation in the n=k step
1+2+…+k+(k+1) = k(k+1)/2 + (k+1)
= (k(k+1) + 2(k+1))/2
=((k+1)(k+2))/2


Counter example: These can be used to prove a claim to be false (rather than true)


Claim: “Every positive integer is the sum of the squares of two integers”
Counter:
1=02+12
2=12+12
3=?


Infamous 1=2 Proof Mistake

  1. a=b
  2. a2=ab
  3. a2-b2=ab-b2
  4. (a-b)(a+b)=b(a-b)
  5. a+b=b
  6. 2b=b
  7. 2=1
The problem is with step 4->5, where we divide both sides by (a-b). However, because a=b, a-b is 0 and you cannot divide by 0. It is the fine print in the method of dividing both sides by a quantity, that the quantity cannot be 0. It is so rare the case we typically ignore it.

And... that's the gist! Of course, there's plenty more proofs, but this would have been (and still is) a beautiful cheat sheet for any discrete mathematics class.
